# Markdown 笔记应用 PRD

## 1. 产品概述

一个完整的 Markdown 笔记应用，支持笔记的创建、编辑、搜索和删除。

## 2. 用户需求

- 创建笔记（标题 + 内容）
- 左右分栏布局：左侧编辑器，右侧实时预览
- 笔记列表展示与搜索
- 代码块语法高亮
- 数据持久化存储

## 3. 技术方案

| 类别 | 技术选型 |
|------|----------|
| 框架 | React + TypeScript + Vite |
| Markdown 解析 | react-markdown |
| 代码高亮 | react-syntax-highlighter |
| 样式 | Tailwind CSS |
| 数据存储 | LocalStorage |

## 4. 功能清单

### 4.1 笔记管理
- [ ] 新建笔记（标题 + 内容）
- [ ] 编辑已有笔记
- [ ] 删除笔记
- [ ] 笔记列表展示

### 4.2 搜索功能
- [ ] 按标题搜索笔记

### 4.3 编辑器
- [ ] 左侧 Markdown 编辑器
- [ ] 右侧实时预览
- [ ] 代码块语法高亮

└────────────┴────────────────────────────────────────┘

## 6. 数据模型

```typescript
interface Note {
  id: string;
  title: string;
  content: string;
  createdAt: number;
  updatedAt: number;
}
```

## 7. 非功能性需求

- 响应式布局，支持不同屏幕尺寸
- 实时预览延迟 < 100ms
- 数据本地存储，刷新页面不丢失
