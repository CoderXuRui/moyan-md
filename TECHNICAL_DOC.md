## 三、数据模型

```typescript
interface Note {
  id: string           // UUID v4
  title: string         // 笔记标题
  content: string      // Markdown 内容
  createdAt: number    // 创建时间戳
  updatedAt: number    // 更新时间戳
}
```

**存储键**: `markdown-notes-v3`

## 七、AI 扩展功能规划

### 7.1 写作增强（优先级：高）

| 功能 | 说明 | 技术方案 |
|------|------|----------|
| AI 自动补全 | 写作时提供智能续写建议 | 调用 LLM API 流式返回 |
| Markdown 语法修正 | 自动检测并修复格式错误 | 正则 + LLM 双重校验 |
| 代码块解释 | 选中代码块，AI 用中文解释功能 | 选区提取 → LLM 翻译 |

### 7.2 智能整理（优先级：高）

| 功能 | 说明 | 技术方案 |
|------|------|----------|
| 自动打标签 | AI 分析内容，自动推荐分类标签 | LLM 提取关键词 → 映射到预设标签池 |
| 相似笔记推荐 | 根据当前内容推荐相关已有笔记 | 向量嵌入 + 余弦相似度 |
| 智能摘要 | 一键生成笔记摘要 | LLM 总结（100 字以内） |
| 语义搜索 | 从关键词搜索升级为自然语言搜索 | 向量数据库 / LLM 重排序 |

### 7.3 内容处理（优先级：中）

| 功能 | 说明 | 技术方案 |
|------|------|----------|
| Markdown 翻译 | 一键中译英 / 英译中 | LLM 多语言翻译 |
| 格式转换 | 转换为微信公众号 / 知乎 / 邮件格式 | LLM 格式化输出 |
| 标题优化 | AI 建议更吸引人的标题 | LLM 生成 3 个备选标题 |
| 可读性分析 | 分析文章难度，给出优化建议 | LLM 评估 + 分数输出 |

### 7.4 生产力工具（优先级：低）

| 功能 | 说明 | 技术方案 |
|------|------|----------|
| 番茄钟 | 内置专注计时器 | 前端计时 + LocalStorage 统计 |
| 写作统计 | 每日/每周字数统计 | LocalStorage 聚合计算 |
| 目标提醒 | 设置写作目标，AI 提醒 | localStorage + 浏览器通知 |

---

### 7.5 推荐优先实现的功能组合

| 方案 | 功能组合 | 适用场景 |
|------|----------|----------|
| 轻量版 | AI 自动补全 + 智能摘要 | 个人写作辅助 |
| 完整版 | 自动标签 + 语义搜索 + 摘要 + 翻译 | 知识管理、个人知识库 |
| 生产力版 | 番茄钟 + 写作统计 + 目标提醒 | 培养写作习惯 |
